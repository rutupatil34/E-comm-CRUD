# 🛒 E-comm CRUD

A simple and interactive **E-commerce CRUD application** built using **HTML, CSS, and JavaScript**.
This project demonstrates the implementation of CRUD operations — **Create, Read, Update, and Delete** — for managing products in an e-commerce interface.

CRUD refers to the basic operations used to manage data: creating new records, reading/displaying data, updating existing records, and deleting records.

---

## 📌 Features

* 🛍️ Add new products
* 👀 Display product cards dynamically
* ✏️ Update existing product details
* 🗑️ Delete products
* 💾 Data stored using Local Storage
* ⚡ Dynamic DOM manipulation

---
